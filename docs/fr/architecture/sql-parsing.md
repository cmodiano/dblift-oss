# Système d'Analyse SQL

**Emplacement** : `core/sql_parser/`

Analyse les scripts SQL pour extraire les instructions et gérer la syntaxe spécifique au dialecte.

## Vue d'Ensemble

Le système d'analyse SQL est responsable de :

- Diviser les scripts SQL en instructions individuelles
- Gérer les délimiteurs spécifiques au dialecte
- Traiter les commentaires et la syntaxe spéciale
- Supporter plusieurs dialectes SQL

Voir la [documentation complète en anglais](../architecture/sql-parsing.md) pour plus de détails.
