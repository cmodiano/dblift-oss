# Système de Configuration

**Emplacement** : `config/`

Gère les paramètres de l'application à partir de plusieurs sources.

## Structure de Configuration

La configuration peut être chargée à partir de :
1. Arguments CLI (priorité la plus élevée)
2. Variables d'environnement
3. Fichier de configuration YAML
4. Valeurs par défaut

Voir la [documentation complète en anglais](../architecture/configuration.md) pour plus de détails.
